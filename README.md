# Club_Codage
Club Codage du lycée Trégey
Dans cet espace, vous trouverez les fichiers qui peuvent servir de modèle pour le codage.
<ul><li>page.html qui reprend la structure de base d'un fichier html avec les principales balises</li>
<li>style.css, qui est associé à page.html et qui assure le réglage du style de cette page (apparence).</li>
<li>formulaire.html qui reprend les éléments d'un formulaire, commeles cases à cocher, les listes de sélection, les champs de saisie, etc.</li>
<li>formulaire.js qui donne quelques idées de code javascript pour animer un peu l'affichage de la page formulaire.html</li></ul>
